# /records-init — records テンプレを生成（新規プロジェクト向け）

## 実行（推奨）

```bash
python .cursor/scripts/ops.py init --project <PROJECT_NAME>
```

例:

```bash
python .cursor/scripts/ops.py init --project trademaster
```

---

## 生成されるもの（例）

- `records/<project>/status.md`
- `records/<project>/<project>_implementation_plan.md`
- `records/<project>/requests/`
- `records/<project>/decisions/_TEMPLATE_decision.md`
- `records/<project>/handover/_TEMPLATE_handover.md`
